Gravar i compartir vídeo del projecte Coldplay

L'ordre de treball és primer fem la presentació amb enllaços tipus Pecha-kucha, la gravem amb OBS, posem els subtítols amb Aegisub. Després combinem vídeo i subtítols amb Handbrake i amb la mateixa eina comprimim el vídeo per sota de 25Mb. Per últim pujem el video a GitHub i afegim video a la web mitjançant codi amb reproductor vídeo html en la nostra web GitHub d'alumne

Primer Pas: Gravar Presentació LibreOffice Impress amb subtítols: OBS, Aegisub i Handbrake

Aquest tutorial et guiarà pas a pas per gravar una presentació de LibreOffice Impress amb OBS, incloent la teva veu, i afegir subtítols utilitzant Aegisub. Aquest procés et permetrà crear vídeos de presentacions amb explicacions i subtítols personalitzats.

1. Preparar OBS per a la gravació de la presentació

Abans de començar la gravació, has de configurar OBS perquè capturi la teva pantalla i el micròfon. Segueix aquests passos:

  • Pas 1: Descarrega i instal·la OBS des de la seva pàgina oficial: obsproject.com.
  • Pas 2: Obre OBS al teu ordinador.
  • Pas 3: Afegeix una font de captura per la teva pantalla:
    • Clica el botó + (Font) a la part inferior esquerra, a la secció "Fonts".
    • Selecciona "Captura de pantalla" i anomena la font, per exemple, "Presentació LibreOffice".
    • Escull la pantalla on tens oberta la presentació de LibreOffice Impress.
  • Pas 4: Afegeix la font del micròfon per capturar la teva veu:
    • A la secció "Fonts", clica el botó + (Font) i selecciona "Dispositiu d'àudio d'entrada".
    • Escull el micròfon que utilitzes per parlar.
  • Pas 5: Si vols capturar tota la pantalla (incloent els enllaços web oberts en la teva presentació), selecciona "Captura de pantalla" en lloc de "Captura de finestra".
  • Pas 6: Ajusta el volum del micròfon i la captura de pantalla per assegurar-te que la teva veu es senti clarament.

2. Com començar a gravar la presentació amb OBS

Un cop configurades les fonts, és hora de començar a gravar la presentació:

  • Pas 1: Abre la teva presentació de LibreOffice Impress i selecciona la diapositiva inicial.
  • Pas 2: A OBS, a la part inferior dreta, clica el botó Iniciar gravació per començar a gravar.
  • Pas 3: Comença a explicar la teva presentació mentre es grava. Recorda que el micròfon ha de captar la teva veu mentre la presentació avança.
  • Pas 4: Quan acabes la presentació, torna a OBS i clica Aturar gravació per finalitzar el vídeo.
  • Pas 5: El vídeo es desarà al directori que hagis configurat a OBS.

3. Crear i sincronitzar subtítols amb Aegisub

Per afegir subtítols al vídeo, utilitzarem Aegisub. A continuació et guiaré pas a pas sobre com fer-ho:

  • Pas 1: Descarrega i instal·la Aegisub des de la seva pàgina oficial: aegisub.org.
  • Pas 2: Obre Aegisub i carrega el teu vídeo:
    • Clica a Archivo > Abrir vídeo i selecciona el vídeo gravat amb OBS.
    • El vídeo es carregarà a la finestra principal de Aegisub.
  • Pas 3: Crear subtítols:
    • A l'àrea de subtítols, veuràs la línia de temps del vídeo. Per cada frase que vulguis subtitular, selecciona el moment exacte en què comença la frase (a la línia de temps) i escriu el subtítol a la columna "Text".
    • Per ajustar el temps de cada subtítol, utilitza els botons de la línia de temps per establir el temps d'inici i fi del subtítol (segons quan es diu la frase).
    • Repetix aquest procés per afegir tots els subtítols del vídeo.
  • Pas 4: Ajustar la sincronització:
    • Reprodueix el vídeo per veure com es veuen els subtítols. Si els temps no coincideixen, ajusta els temps d'inici i final fins que estiguin ben sincronitzats.
  • Pas 5: Exportar els subtítols:
    • Quan hagis afegit tots els subtítols i estiguin sincronitzats, clica a Archivo > Guardar com i selecciona el format .srt (subtítols).

4. Incrustar subtítols al vídeo

Per incrustar els subtítols al teu vídeo, pots utilitzar un programa com HandBrake. Aquí tens els passos per fer-ho:

  • Pas 1: Descarrega i instal·la HandBrake des de la seva pàgina oficial: handbrake.fr.
  • Pas 2: Obre HandBrake i carrega el vídeo gravat amb OBS.
  • Pas 3: A la secció "Subtítols", selecciona Afegeix un fitxer de subtítols i selecciona el fitxer .srt creat amb Aegisub.
  • Pas 4: Clica a Iniciar codificació per crear un nou vídeo amb els subtítols incrustats.
  • Pas 5: El vídeo amb els subtítols incrustats es guardarà a la ubicació especificada.

5. Finalitzar i compartir el vídeo

Un cop tinguis el teu vídeo amb la presentació gravada i els subtítols incrustats, pots compartir-lo en línia:

  • Pas 1: Revisa el vídeo per assegurar-te que els subtítols es veuen correctament i estan sincronitzats amb la teva veu.
  • Pas 2: Pujar el vídeo a una plataforma com YouTube, Vimeo, o qualsevol altre lloc on vulguis compartir la presentació.
  • Pas 3: Comparteix el teu vídeo amb els altres!

Consells addicionals:

  • Pots modificar la font i el disseny dels subtítols a Aegisub per personalitzar-los segons el teu gust.
  • Si necessites fer canvis als subtítols després de la gravació, pots editar el fitxer .srt directament a Aegisub.
  • Per una millor qualitat de gravació, ajusta les configuracions d'OBS a "Sortida" per ajustar la resolució i taxa de bits.

Amb aquests passos, hauràs creat una presentació gravada, amb veu i subtítols sincronitzats. Bona sort amb la teva creació!

Segon pas: Comprimir vídeo a menys de 25MB i penjar-lo a GitHub Pages

Aquest tutorial et guiarà per comprimir un vídeo utilitzant HandBrake i després pujar-lo a GitHub Pages per a compartir-lo. Seguirem els passos següents per aconseguir-ho.

1. Comprimir el vídeo amb HandBrake

Després de gravar la teva presentació amb OBS i afegir subtítols amb Aegisub, segurament voldràs comprimir el vídeo per assegurar-te que el fitxer no sigui massa gran. Aquí tens els passos per comprimir el teu vídeo a menys de 25 MB utilitzant HandBrake:

  • Pas 1: Descarrega i instal·la HandBrake des de la seva pàgina oficial: handbrake.fr.
  • Pas 2: Obre HandBrake i carrega el teu vídeo gravat amb OBS:
    • Clica a Open Source i selecciona el fitxer del vídeo que vols comprimir.
  • Pas 3: Ajusta les opcions de compressió per reduir la mida del vídeo:
    • Tria el perfil Fast 1080p30 a la secció Presets, que té una bona compressió per vídeos de qualitat mitjana.
    • A la secció Video, assegura't que el còdec sigui H.264 (x264).
    • Redueix la qualitat a un valor de RF: 28-30 per aconseguir una bona compressió sense perdre massa qualitat.
    • Si la mida encara és gran, pots provar de reduir la resolució a 720p (1280x720) per ajudar a reduir encara més la mida del fitxer.
  • Pas 4: Afegeix el nom del fitxer d'output:
    • A la secció Save As, tria la ubicació on vols guardar el fitxer comprimit i posa-li un nom.
  • Pas 5: Comença la compressió:
    • Clica el botó Start Encode per començar la compressió del vídeo.
    • Un cop la compressió hagi acabat, verifica la mida del vídeo per assegurar-te que és inferior a 25 MB.

2. Penjar el vídeo a GitHub Pages

Un cop tens el teu vídeo comprimit, pots penjar-lo a GitHub Pages. A continuació et guiaré sobre com fer-ho:

  • Pas 1: Crea un repositori a GitHub:
    • Accedeix a GitHub i crea un repositori nou. Pots anomenar-lo, per exemple, "presentacio-video".
    • Marca l'opció Initialize this repository with a README i clica a Create repository.
  • Pas 2: Puja el vídeo comprimit al repositori:
    • A la pàgina del repositori creat, clica a Upload files.
    • Afegeix el fitxer de vídeo comprimit que acabes de crear.
    • Un cop seleccionat el fitxer, clica a Commit changes per pujar el fitxer al repositori.
  • Pas 3: Habilita GitHub Pages:
    • A la pàgina del repositori, ves a la secció Settings i baixa fins a la part de GitHub Pages.
    • En la secció Source, selecciona main branch com a font per a GitHub Pages.
    • Després de configurar-ho, GitHub et proporcionarà una URL pública on el teu lloc web serà accessible, com ara https://el_teu_usuari.github.io/presentacio-video/.
  • Pas 4: Crea un fitxer HTML per mostrar el vídeo:
    • Al repositori, clica a Create new file.
    • Anomena el fitxer com index.html i afegeix el següent codi per incrustar el vídeo:
    <!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Presentació de vídeo</title>
</head>
<body>
    <h1>Presentació de la meva presentació</h1>
    <video controls>
        <source src="nom_del_video_comprimit.mp4" type="video/mp4">
        El teu navegador no suporta el format de vídeo MP4.
    </video>
</body>
</html>
  • Pas 5: Guarda els canvis i visita la teva URL de GitHub Pages per veure el vídeo incrustat.

Consells addicionals:

  • Assegura't que el vídeo no superi els 25 MB. Si és més gran, intenta reduir la qualitat o la resolució fins a aconseguir la mida desitjada.
  • GitHub Pages és una eina poderosa per compartir vídeos petits i contingut estàtic de manera senzilla i gratuïta.
  • Si tens més fitxers (com imatges o documents), pots afegir-los al mateix repositori i fer-los accessibles des del mateix lloc web de GitHub Pages.

Amb aquests passos, podràs comprimir el teu vídeo i penjar-lo fàcilment a GitHub Pages per compartir-lo amb qui vulguis.
